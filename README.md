# 🔗 Supply Chain Disruption Analyzer

## Overview
A data analytics project that tracks and analyzes global supply chain 
disruptions using real Federal Reserve economic data from 2018 to 2026.
Built a weighted risk scoring model across 4 economic indicators and 
visualized insights across 5 industries.

## 📊 Live Dashboards
- 📌 Executive Summary: https://public.tableau.com/views/SupplychainExecutiveSummary/SupplyChainSummary
- 📌 Detailed Analysis: https://public.tableau.com/shared/GKJZPBBCQ

## 🛠 Tools Used
- Python (Pandas, Matplotlib, FRED API)
- Tableau Public
- Jupyter Notebook
- Excel

## 🔍 Key Findings
- June 2022 was the peak disruption month — oil hit $114/barrel
- Automotive industry is 35% more exposed than Healthcare
- Current risk score of 72 is 3x higher than pre-COVID baseline
- COVID caused unemployment to spike from 3.5% to 14.7% in just 2 months

## 📁 Data Sources
- Federal Reserve FRED API
- Indicators: Crude Oil Price, Producer Price Index, 
  Consumer Price Index, Unemployment Rate
- Date range: January 2018 to present
